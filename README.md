# CI4 Product Management System (Practical Exam)

A secure Product CRUD application built with CodeIgniter 4 and Bootstrap 5.

## Features
- **Authentication**: Secure Login/Register with Session protection.
- **Middleware**: Route protection via AuthGuard and GuestGuard.
- **CRUD**: Full Create, Read, Update, and Delete for Product records.
- **UI**: Modern, responsive dashboard using Bootstrap 5.

## Setup
1. Clone the repo.
2. Run `composer install`.
3. Create a database named `ci4_exam_db`.
4. Import the SQL file (if provided).
5. Rename `env` to `.env` and set your database credentials.
